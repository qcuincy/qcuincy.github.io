Project 3

Deep Embeddings for Visual Search.

Overview

  • Objective: Enhance visual searching through deep embedding machine-learning techniques.
  • Focus: Improved feature representation of clothing items for versatile visual searches.
  • Method: Combined transfer learning and contrastive learning loss.
  • Outcome: Model effectively groups clothing items based on patterns and styles.
  • Implication: Potential to reshape the fashion industry by aiding customers in finding matching clothing items.

Approach

  • Strategy: Utilized transfer learning and contrastive learning loss.
  • Transfer Learning: Employed ResNet-34 model as backbone network for its pre-trained features.
  • Key Innovation: Contrastive learning loss integration for intricate feature representations.
  • Result: Enhanced accuracy in visual search model by refining feature extraction process.

Approach - Contrastive Learning Loss

  • Technique: Contrastive learning loss function applied.
  • Principle: Loss function maps similar samples close, dissimilar samples apart in feature space.
  • Benefit: Captured unique attributes of clothing patterns and styles effectively.
  • Inspiration: Insights from [22] and [23] used to harness contrastive learning loss potential.
  • Impact: Enabled model to excel in detecting clothing items from different angles and viewpoints.

Challenges

  • Complexity: Project faced several challenges despite promising results.
  • Augmentation: Model performance optimization through augmentation techniques.
  • Balance: Ensured augmentation enhanced robustness without causing overfitting.
  • Parameter Tuning: Contrastive learning loss integration required careful parameter adjustment.
  • Struggle: Finding the right balance between encouraging proximity for similar samples and maintaining separation for dissimilar ones highlighted project complexity.
