Project 4

Solar Panel Dust Detection.

Overview

  • Objective: Implement an object detection system for identifying solar panels in drone footage.
  • Focus: Utilize YOLO (You Only Look Once) as the primary object detection model.
  • Method: Conducted extensive research to select the most suitable pretrained object detection model. Preprocessed data by resizing images and established train-test-validation splits. Created visualizations and a GUI for model demonstration.
  • Outcome: Successfully trained and implemented the YOLO object detection model for solar panel identification.
  • Implication: The project demonstrates the effectiveness of YOLO for real-world applications in renewable energy infrastructure monitoring.

Approach

  • Strategy: Employed a combination of pre-trained models, data preprocessing, and custom visualization tools to achieve accurate solar panel detection.
  • Transfer Learning: Leveraged transfer learning techniques to adapt the YOLO model to the specific task of solar panel detection.
  • Key Innovation: Developed a GUI for user-friendly model demonstration and comparison.
  • Result: Achieved accurate and efficient detection of solar panels in various environmental conditions.

Approach - YOLO

  • Technique: Implemented the YOLO object detection algorithm, which processes the entire image in a single forward pass through the neural network.
  • Principle: YOLO divides the image into a grid and predicts bounding boxes and class probabilities for each grid cell.
  • Benefit: YOLO provides real-time object detection with high accuracy, making it ideal for applications like solar panel identification.
  • Inspiration: YOLO's speed and accuracy were essential for achieving timely and reliable detection of solar panels in dynamic environments.

Challenges

  • Complexity: Balancing model complexity and computational resources was crucial to ensure real-time performance.
  • Augmentation: Data augmentation techniques were applied to enhance the model's ability to generalize across diverse environmental conditions.
  • Balance: Striking a balance between precision and recall was a challenge due to the importance of accurate solar panel identification.
  • Parameter Tuning: Fine-tuning hyperparameters was necessary to optimize model performance for the specific detection task.
  • Struggle: Overcoming the computational demands of real-time detection while maintaining accuracy posed a significant challenge, especially for drone footage with varying lighting conditions.
