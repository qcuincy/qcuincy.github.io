Project 7

Reducing Latency & Improving Accuracy in Dynamic Hand Gesture Recognition.

Overview:

  • Developed an advanced system for early classification of dynamic hand gestures in virtual reality (VR) environments, focusing on latency reduction and improved accuracy.

Key Contributions:

  • Engineered discrete features from 3D hand landmark coordinates, encompassing moving direction, palm orientation, and hand pose
  • Implemented and trained Transformer models to predict future hand states, enabling early gesture recognition
  • Developed a novel classification system using lookup tables and sequence similarity measures, specifically employing Dynamic Time Warping (DTW)
  • Conducted comprehensive evaluations comparing multi-step and autoregressive Transformer models
  • Explored the balance between prediction accuracy and latency reduction in VR interactions

Technical Implementations:

  • Created discrete time series sequences from hand tracking data for efficient processing
  • Utilised Transformer architecture for its ability to capture long-range dependencies in sequential data
  • Implemented DTW for robust classification of variable-length gesture sequences
  • Developed evaluation metrics including Transformer prediction accuracy and Cross-Entropy Loss

Research Implications:

  • Contributed to the field of gesture recognition in VR, addressing critical issues of latency and accuracy
  • Demonstrated the potential of Transformer models in early classification of dynamic gestures
  • Provided insights into the trade-offs between prediction accuracy and real-time performance in VR environments

Skills Demonstrated:

  • Advanced machine learning model design and implementation
  • Feature engineering for 3D spatial data
  • Time series analysis and prediction
  • Performance optimisation for real-time systems
  • Scientific research methodology and evaluation

Future Directions:

  • Identified potential for exploring alternative model architectures to further improve performance
  • Suggested methods for enhancing data quality and incorporating additional sensor data for richer context in VR interactions
